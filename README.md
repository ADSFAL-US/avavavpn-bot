# Avava VPN Bot

Telegram бот для управления подписками Avava VPN.

## Возможности

- 📋 Выбор тарифа через встроенное меню
- 📊 Панель администратора со статистикой
- 👥 Управление пользователями (бан/разбан, админки)
- ⚡ Настройка скорости для поддерживаемых тарифов
- 📝 Логирование действий администраторов
- 🔒 Система банов пользователей

## Установка

### 1. Установите зависимости

```bash
pip install python-telegram-bot
```

### 2. Настройте переменные окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Или создайте файл `.env`:

```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=123456789
DATABASE_PATH=avava_vpn.db
VPN_SERVER_ADDRESS=vpn.avava.local
VPN_PORT=1194
VPN_PROTOCOL=udp
```

### 3. Получите токен бота

1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### 4. Запустите бота

```bash
python bot.py
```

## Управление администратором

Для назначения себя админом:

1. Запустите бота
2. Отправьте `/start`
3. В базе данных установите `is_admin = 1` для вашего user_id:

```sql
UPDATE users SET is_admin = 1 WHERE user_id = ВАШ_ID;
```

Или добавьте ваш ID в `.env` в `ADMIN_IDS`.

## Команды бота

### Пользовательские команды:
- `/start` - начать работу
- `/tariffs` - просмотреть тарифы
- `/my_subscription` - информация о подписке
- `/speed_upgrade` - увеличить скорость
- `/support` - поддержка

### Административные команды:
- `/admin` - панель администратора
- `/stats` - статистика

## База данных

Используется SQLite. Файл базы данных создаётся автоматически при первом запуске.

### Таблицы:
- `users` - информация о пользователях
- `subscriptions` - подписки пользователей
- `vpn_connections` - активные подключения VPN
- `speed_upgrades` - запросы на повышение скорости
- `admin_log` - логи действий администраторов
